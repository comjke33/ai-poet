# ai-poet
인공지능 시인 / streamlit으로 구현

- OPENAI gpt-4o-mini API 기반
- langchain
- streamlit

### TEST
1. `git clone [git 주소]`
2. `python -m venv venv`
3. `cd venv/Scripts`
4. `.\activate`
5. `pip install -r requirements.txt`
6. `streamlit run main.py`
